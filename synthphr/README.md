# Synthetic PHR


Complete Personal Health Record (“PHR”):
* Conditions: SNOMED-CT CORE (5,316 Findings) Devices: FDA’s Approved Devices
* Drugs: FDA’s NDC
* Treatments: SNOMED-CT
* Lab Results: LOINC
* JSON, SQLite3 output. Could generate HL7, FHIR.
